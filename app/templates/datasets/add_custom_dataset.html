数据格式说明

问答题格式 (QA) - JSONL文件

示例1：
{ "system": "你是一个专业的助手", "query": "什么是人工智能？", "response": "人工智能是计算机科学的一个分支..." }
示例2：
{ "query": "今天天气怎么样？", "response": "今天天气晴朗，温度适宜。" }

system: 系统提示（可选字段）
query: 问题（必须字段）
response: 正确回答（必须字段）

选择题格式 (MCQ) - CSV文件

CSV格式示例：
id,question,A,B,C,D,answer 1,以下哪种元素最多,氢,氧,碳,氮,A 2,地球的卫星是,太阳,月球,火星,金星,B

id: 序号（可选字段）
question: 问题（必须字段）
A,B,C,D等: 选项（最多支持10个选项，A-J）
answer: 正确选项（如A,B,C,D）

自定义格式 (CUSTOM) - JSONL文件

示例1（包含历史对话）：
{ "history": [ {"user": "你好", "assistant": "你好！有什么可以帮助你的吗？"} ], "question": "请介绍一下机器学习", "answer": "机器学习是人工智能的一个重要分支..." }
示例2（简单格式）：
{ "question": "什么是深度学习？", "answer": "深度学习是机器学习的一个子领域..." }

history: 历史对话信息，首轮不用填，有的话使用list格式
question: 用户问题
answer: 正确答案（必须字段）

RAG格式 (RAG) - JSONL文件

示例1（完整格式）：
{ "user_input": "第一届奥运会是什么时候举行的？", "retrieved_contexts": [ "第一届现代奥运会于1896年4月6日到4月15日在希腊雅典举行。", "这届奥运会共有来自14个国家的241名运动员参加。" ], "response": "第一届现代奥运会于1896年4月6日举行。", "reference": "第一届现代奥运会于1896年4月6日在希腊雅典开幕。" }
示例2（简化格式）：
{ "user_input": "什么是人工智能？", "reference": "人工智能是计算机科学的一个分支..." }

user_input: 用户问题（必须字段）
retrieved_contexts: 检索到的上下文列表（可选字段, 不填需要通过jinja2提供）
response: 模型回答（可选字段, 不填需要通过jinja2提供）
reference: 标准答案（必须字段）
